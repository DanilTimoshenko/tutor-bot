# Мой сайт

Одностраничный сайт-заготовка.

## Как открыть в Cursor

1. В Cursor: **Файл → Открыть папку** (или `Cmd+O`).
2. Выберите папку **my-website**.
3. Редактируйте `index.html`, `styles.css` и `script.js`.

## Как посмотреть сайт в браузере

- Откройте файл `index.html` в браузере двойным щелчком, или  
- Запустите локальный сервер в папке `my-website`, например:
  - `python3 -m http.server 8000` — затем откройте http://localhost:8000
  - или расширение Live Server в VS Code/Cursor

## Структура

- `index.html` — разметка страницы
- `styles.css` — стили
- `script.js` — скрипты
