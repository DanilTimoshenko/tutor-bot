# Фото решений к заданиям ЕГЭ

Положи сюда изображения решений, например:
- `1.png` — для задания 1
- `2.png` — для задания 2

В JSON или в БД у задания укажи поле **solution_image**:
- путь к файлу: `ege_images/1.png`
- или URL картинки: `https://example.com/solution1.png`

При открытии задания в боте после текста будет отправлено фото с подписью «Решение (рисунок)».
